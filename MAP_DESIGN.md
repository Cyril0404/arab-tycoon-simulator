## 一、设计理念

### 核心理念
- **长期目标**：让玩家玩6个月~1年以上
- **核心驱动**：稀缺资源竞争 + 社交攀比 + 收集成就感
- **参考游戏**：大富翁 + 地产大亨 + 万国觉醒 + COC

### 核心循环
```
购买地块 → 升级地块 → 收集连块 → 称霸城市
    ↓
收租产出 → 购买更多地块 → 参与战争
    ↓
加入联盟 → 联盟战 → 统治地区 → 成为世界霸主
```

## 二、世界分层架构

```
┌─────────────────────────────────────────┐
│          第5层：世界（World）             │  ← 终极目标，全服唯一
├─────────────────────────────────────────┤
│          第4层：地区（Region）            │  ← 多个城市组成
├─────────────────────────────────────────┤
│          第3层：城市（City）              │  ← 多个街区组成
├─────────────────────────────────────────┤
│          第2层：街区（District）          │  ← 多个地块相连
├─────────────────────────────────────────┤
│          第1层：地块（Plot）              │  ← 最小单位，可购买
└─────────────────────────────────────────┘
```

### 各层解锁条件

| 层级 | 名称 | 解锁条件 | 奖励 |
|------|------|---------|------|
| L1 | 地块 | 游戏开始即可购买 | 基础收租 |
| L2 | 街区 | 持有相邻3块地块 | 街区BUFF + 收租×1.5 |
| L3 | 城市 | 集齐某城市所有地块 | 城市称霸称号 + 城市BUFF |
| L4 | 地区 | 称霸区域内所有城市 | 地区霸主称号 |
| L5 | 世界 | 全球制霸 | 「世界之王」永久称号 + 全服广播 |

## 七、城市战争系统

### 7.1 宣战流程

```
1. 玩家点击「宣战」按钮（需持有该城市至少1块）
   ↓
2. 选择目标城市（必须已拥有该城市至少1块）
   ↓
3. 购买「战争债券」（每次战争需消耗）
   ↓
4. 48小时备战期
   ↓
5. 战争开始（每日结算）
   ↓
6. 48小时后结算
```

### 7.2 战争机制

**驻扎：**
- 战争期间，可向对方城市任意地块发起「驻扎」
- 驻扎 = 在该地块上「插旗」，每日扣对方该地块收租的10%
- 驻扎无人数上限
- 每次驻扎消耗1个「战争债券」

**反击：**
- 地块所有者可用「战争债券」反击
- 每次反击 = 驱逐对方1个驻扎单位
- 反击不消耗额外债券（每地块每日限1次反击）

### 7.3 战争债券

| 债券价格 | 用途 |
|---------|------|
| 10,000 AED/个 | 驻扎1次 |
| 50,000 AED/5个 | 批量购买 |

### 7.4 战争奖励

| 结果 | 进攻方 | 防守方 |
|------|--------|--------|
| 进攻方胜 | 抢走该城市最贵的1块地（直接过户）| 所有地块降级1次（收租-20%）|
| 进攻方败 | 战争债券全部没收（50%折算为全服奖励池）| 全城所有者收租+20%（守土有功）|

### 12.3 购买/升级流程

```
用户点击地块
  ↓
检查：是否已拥有？→ 已拥有 → 提示"这是你的地"
  ↓ 未拥有
检查：价格 ≤ 现金？ → 不足 → 提示"余额不足"
  ↓ 充足
弹出购买确认面板
  ↓ 用户确认
写入 Firestore: plots/{plotId}.ownerId = userId
  ↓
更新玩家 localStorage
  ↓
播放购买成功动画
  ↓
计算连块奖励 → 更新收租倍率显示
```

### 12.4 收租计算

```javascript
function calculateDailyRent(plotId, userId) {
  const plot = getPlot(plotId)
  const user = getUser(userId)
  const upgrades = getUpgrades(plotId)

  // 基础收租
  let rent = plot.dailyRent

  // 升级倍率
  rent *= upgrades.level_multplier  // Lv1=1, Lv2=1.5, Lv3=2 ...

  // 城市BUFF
  rent *= CITY_BUFF[plot.city]  // Dubai=1.1, Riyadh=1.15 ...

  // 连块加成
  const adjacentOwned = plot.adjacent.filter(id => getOwner(id) === userId)
  if (adjacentOwned.length >= 2) rent *= 1.5
  if (adjacentOwned.length >= 4) rent *= 2.0

  // 联盟加成
  if (user.allianceId) rent *= 1.1

  return Math.floor(rent)
}
```
