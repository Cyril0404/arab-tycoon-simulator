# 产品需求文档（PRD）

> **项目**: Arab Tycoon Simulator（阿拉伯土豪模拟器）
> **版本**: v1.0
> **最后更新**: 2026-03-29
> **作者**: 开发团队

---

## 1. 项目概述

### 1.1 项目背景

Arab Tycoon Simulator 是一款面向中东阿拉伯国家用户的 H5 休闲炫富社交游戏。玩家从 1,000 AED（迪拉姆）起步，通过打工、经商、投资等行为积累财富，购买豪车、豪宅、宠物、奢侈品，最终目标是成为"中东首富"。

游戏的核心卖点是**社交攀比**和**炫富传播**，鼓励玩家将财富截图分享到 WhatsApp、Instagram 等社交平台，通过排行榜机制激发竞争欲望。

### 1.2 项目愿景

> *"Start with 1,000 AED, become the richest man in the Middle East!"*

- 让普通用户体验一把"中东土豪"的快感
- 满足用户的炫富心理和社交攀比需求
- 通过病毒式传播（WhatsApp 分享）实现低成本获客

### 1.3 目标平台

| 平台 | 支持情况 | 说明 |
|------|----------|------|
| WhatsApp 内置浏览器 | ✅ 完美支持 | 核心使用场景 |
| 手机 Chrome | ✅ 完美支持 | Android 主流浏览器 |
| 手机 Safari | ✅ 完美支持 | iOS 主流浏览器 |
| 桌面浏览器 | ⚠️ 基本支持 | 仅作开发调试用 |
| 微信内置浏览器 | ❌ 不支持 | 暂不考虑 |

### 1.4 MVP 范围 vs 后期迭代

**MVP（v1.0）必须包含：**
- 启动页 + 新手引导
- 核心赚钱/消费玩法
- 身份等级系统
- 炫富商品购买
- 排行榜（总资产/豪车/豪宅/宠物）
- 好友系统（添加、拜访、送礼、比富 PK）
- WhatsApp 分享海报

**v1.1 迭代计划：**
- 成就系统
- 每日任务
- 石油投资彩蛋（亏损/盈利随机事件）
- 阿拉伯语 RTL 支持
- 多人联机 PK 房间

**v2.0 远期规划：**
- 公会/家族系统
- 投资理财模拟（股票、房产）
- 皮肤/头像自定义
- 广告变现（插屏广告）

## 4. 游戏经济系统

### 4.1 货币体系

| 货币类型 | 符号 | 用途 |
|----------|------|------|
| 迪拉姆 | AED | 唯一货币，所有交易使用 AED |

### 4.2 赚钱行为详细参数

| 行为 | 英文名 | 基础收益 | 冷却时间 | 说明 |
|------|--------|----------|----------|------|
| 打工 | Work | +1,000 AED | 60 秒 | 最基础的赚钱方式 |
| 做生意 | Business | +10,000 AED | 600 秒（10分钟） | 收益较高 |
| 投资石油 | Oil Investment | +1,000,000 AED（成功）/-500,000 AED（失败） | 3600 秒（1小时） | 50% 概率盈利 2x，50% 概率亏损 0.5x |

**平衡性说明：**
- 打工：低收入高频次，保证玩家随时有事可做，适合早期
- 做生意：中收入中频次，需要玩家有耐心等待
- 投资石油：高风险高回报，符合"土豪"人设，同时保持刺激感

**投资石油概率算法：**
```javascript
function investOil() {
  const roll = Math.random();
  if (roll >= 0.5) {
    return { success: true, amount: 1_000_000 };
  } else {
    return { success: false, amount: -500_000 };
  }
}
```

### 4.3 消费行为（购买装备）

| 行为 | 说明 |
|------|------|
| 购买装备 | 从商店购买豪车/豪宅/宠物/奢侈品 |
| 购买后装备自动计入总资产 | 总资产 = 现金 + 所有装备当前价值 |
| 装备可重复购买 | 同一装备可买多份（限量版例外） |
| 退货 | 购买后 24 小时内可原价退货（扣除 10% 手续费） |

### 4.4 总资产计算公式

```
totalWealth = cash + sum(ownedItems.itemCurrentValue)

其中：
- cash = 可支配现金（未花费的 AED）
- ownedItems = 用户已购买的所有装备
- itemCurrentValue = 装备当前价值（奢侈品每年贬值 5%，豪车/豪宅/宠物不贬值）
```

**身份等级门槛（自动计算）：**

| 等级序号 | 等级名称（英文） | 等级名称（中文） | 总资产门槛（AED） | 解锁内容 |
|----------|----------------|----------------|-------------------|----------|
| 1 | Commoner | 平民 | 0 | 基础打工、做生意 |
| 2 | Small Boss | 小老板 | 100,000 | 解锁豪车Tab |
| 3 | Rich Merchant | 富商 | 1,000,000 | 解锁豪宅Tab |
| 4 | Tycoon | 超级富豪 | 100,000,000 | 解锁宠物Tab、PK功能 |
| 5 | Middle East's Richest | 中东首富 | 10,000,000,000 | 全服公告、解锁限量版装备 |

### 4.5 经济平衡性设计原则

1. **前 10 分钟**：玩家通过打工可以赚到 ~10,000 AED，体验快速成长
2. **第 1 小时**：玩家通过做生意和打工可以赚到 ~100,000 AED，解锁"小老板"
3. **第 1 天**：活跃玩家通过各种手段可达 ~10,000,000 AED
4. **防止通货膨胀**：装备价格不设通胀系数，保持固定
5. **防止刷钱**：所有赚钱行为均有服务器端冷却时间验证

|------|------|

### 7.2 好友互动行为

| 行为 | 描述 | 奖励 |
|------|------|------|
| 拜访 | 进入好友主页，浏览其装备 | 每日首次拜访获得 100 AED |
| 点赞 | 给好友的装备点赞 | 每日可点赞 10 次，点赞方得 50 AED |

### 7.3 PK 对战规则

```
PK流程：
1. 发起方点击"PK挑战" → 好友收到通知
2. 好友接受挑战（24小时内，否则超时失效）
3. 系统比较双方 totalWealth
4. 胜利条件：totalWealth 高者胜出
5. 平局：双方均为"小土豪"称号（时长翻倍）
6. 奖励：胜利方获得"小土豪"临时称号（24h）
```

## 8. 分享机制设计

### 8.1 炫富海报生成

**海报内容：**
- 背景：金色渐变 + 迪拜天际线剪影
- 顶部：游戏 Logo
- 中央：大字显示总资产（"AED 1.5亿"）
- 下方：身份等级 + 排名（"排名全服第3位"）
- 底部：二维码（游戏链接） + "Play Now"

**尺寸：** 1080 × 1920 px（适合 Instagram Story / WhatsApp Status）

### 8.2 分享渠道

| 渠道 | 分享类型 | 技术实现 |
|------|----------|----------|
| WhatsApp | 链接 + 海报图片 | Web Share API + Canvas 截图 |
