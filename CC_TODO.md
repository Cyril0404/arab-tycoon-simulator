### 3. 收入速度加成（买车/房/地/石油加收益%）

**加成表：**
| 已购资产类型 | 收入加成 |
|------|------|
| 任意车辆 | +10% |
| 任意房产 | +15% |
| 沙漠领土 | +20% |
| 石油开采权 | +30% |

**实现方式：**
- 用户数据结构加 `incomeBonusPercent: 0`
- 每次购买资产后，重新计算加成总和
- 打工收益 = 基础收益 × (1 + incomeBonusPercent%)
